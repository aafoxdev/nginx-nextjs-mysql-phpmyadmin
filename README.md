# 1. NginxとDockerを利用した結構実用的なサーバー構築

Dockerを用いて、コマンド2つで本格的なWebサーバーを構築します。
具体的には、次のコンテナが立ち上がり、機能性抜群のWebサーバーが仮想的に作成されます。
- プロキシサーバーとしてNginxコンテナ
- FE・BE両対応のNextJSコンテナ
- データベースとしてMySQLコンテナ
- データベースの管理としてPHPMyadmin
LAMPとは少し違うモダンな開発環境です...よね？
